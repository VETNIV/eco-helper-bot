Eco Assistant Bot

Prototype for Hackathon IT TECH 2025

Description

This repository contains a simple prototype of the "Ecological Assistant" chatbot.
The bot provides users with ecological advice, recycling points and eco-tips.

This prototype is created for educational and demonstration purposes.

Features

Personalized eco-tips

Simple command-based interface

Extensible architecture

Ready for integration with maps and databases

Technologies

Python 3

Aiogram (Telegram bot framework)

JSON for simple local data storage
How to run
pip install -r requirements.txt
python bot/main.py
